# #30DayMapChallenge

W ramach edycji [**#30DayMapChallenge 2025**](https://30daymapchallenge.com/) postanowiłam stworzyć i zebrać wszystkie moje mapy w jednym miejscu.

#30DayMapChallenge to coroczne wyzwanie polegające na codziennym tworzeniu map przez cały listopad, każda z nich na inny, z góry określony temat.
Format, narzędzia i źródła danych są całkowicie dowolne, co daje dużą swobodę twórczą.
